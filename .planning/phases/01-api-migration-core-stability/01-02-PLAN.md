  <action>
    Convert the following 5 methods from synchronous to async per D-11. Each method keeps its existing logic but replaces sync File I/O with async equivalents. Add `using System.Threading.Tasks;` to the using block.

    **1. WriteFile -> WriteFileAsync**
    ```csharp
    public async Task<string> WriteFileAsync(string filename, string content)
    {
        if (!m_Initialized) return "[Error]: Memory system not initialized.";
        if (!ValidateFilename(filename, out string error)) return error;

        string path = Path.Combine(m_CityDir, filename);
        if (!File.Exists(path))
            return $"[Error]: File '{filename}' does not exist. Use create_memory_file to create new files.";

        await File.WriteAllTextAsync(path, content).ConfigureAwait(false);
        Mod.Log.Info($"[NarrativeMemorySystem] Wrote {content.Length} chars to {filename}");
        return $"Successfully wrote {content.Length} characters to {filename}.";
    }
    ```

    **2. AppendToLog -> AppendToLogAsync**
    Same logic as current `AppendToLog` but:
    - Signature: `public async Task<string> AppendToLogAsync(string entry)`
    - Replace `File.ReadAllText(path)` with `await File.ReadAllTextAsync(path).ConfigureAwait(false)`
    - Replace `File.WriteAllText(path, updated)` with `await File.WriteAllTextAsync(path, updated).ConfigureAwait(false)`
    - Replace `RotateNarrativeLog(path)` call with `await RotateNarrativeLogAsync(path).ConfigureAwait(false)` (also convert RotateNarrativeLog to async — see below)

    **3. SaveChatSession -> SaveChatSessionAsync**
    ```csharp
    public async Task SaveChatSessionAsync(string transcriptMarkdown)
    {
        if (!m_Initialized) return;
        try
        {
            string histDir = Path.Combine(m_CityDir, "chat-history");
            Directory.CreateDirectory(histDir);
            string filename = $"session-{m_SessionNumber:D3}.md";
            string path = Path.Combine(histDir, filename);
            await File.WriteAllTextAsync(path, transcriptMarkdown).ConfigureAwait(false);
            Mod.Log.Info($"[NarrativeMemorySystem] Saved chat session: {filename}");
            await PruneChatHistoryAsync(histDir).ConfigureAwait(false);
        }
        catch (Exception ex)
        {
            Mod.Log.Error($"[NarrativeMemorySystem] SaveChatSession failed: {ex.Message}");
        }
    }
    ```

    **4. CreateFile -> CreateFileAsync**
    Same logic as current `CreateFile` but:
    - Signature: `public async Task<string> CreateFileAsync(string filename, string content)`
    - Replace `File.WriteAllText(path, content)` with `await File.WriteAllTextAsync(path, content).ConfigureAwait(false)`

    **5. DeleteFile -> DeleteFileAsync**
    - Signature: `public async Task<string> DeleteFileAsync(string filename)`
    - File.Delete is not async in .NET Standard 2.1. Wrap it: `await Task.Run(() => File.Delete(path)).ConfigureAwait(false);`

    **6. Convert private helpers to async:**
    - `RotateNarrativeLog` -> `RotateNarrativeLogAsync`: replace File.ReadAllText/WriteAllText with async equivalents
    - `PruneChatHistory` -> `PruneChatHistoryAsync`: File.Delete calls wrapped in Task.Run

    **7. Keep synchronous methods unchanged:**
    - `ReadFile` — sync reads are fast, called from tool dispatch on thread pool
    - `ListFiles` — directory listing, fast
    - `GetAlwaysInjectedContext` — sync reads, called from RunRequestAsync (already on thread pool)
    - `LoadLatestChatSession` — called once at startup from main thread before any async work
    - `Initialize` — called once at startup, scaffolding is a one-time cost
    - `ChatHistoryToMarkdown` — pure function, no I/O
    - `ParseChatSession` — pure function, no I/O

    **Per D-12:** All async methods must have their own try/catch. Failures are logged via `Mod.Log.Error(...)` and swallowed — the method returns an error string (for Task<string> methods) or silently returns (for Task methods). No exceptions propagate to callers.
