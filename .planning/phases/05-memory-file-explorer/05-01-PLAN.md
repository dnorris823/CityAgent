**Step 4: Implement the four handler methods** (add as new private methods in the class):

```csharp
private void OnRefreshMemoryFiles()
{
    string json = m_MemorySystem.ListFiles();
    m_MemoryFilesJson.Update(json);
}

private void OnReadMemoryFile(string filename)
{
    m_MemoryOpResult.Update("");  // Reset before operation (loading state)
    string content = m_MemorySystem.ReadFile(filename);
    m_MemoryOpResult.Update(content);
}

private void OnWriteMemoryFile(string filename, string content)
{
    m_MemoryOpResult.Update("");  // Reset before operation
    string result = m_MemorySystem.WriteFile(filename, content);
    m_MemoryOpResult.Update(result);
}

private void OnDeleteMemoryFile(string filename)
{
    m_MemoryOpResult.Update("");  // Reset before operation
    string result = m_MemorySystem.DeleteFile(filename);
    m_MemoryOpResult.Update(result);
}
```

All handlers execute synchronously on the main thread (per UI-SPEC writeMemoryFile thread
strategy — files are small markdown, sub-50KB). Each handler resets `memoryOpResult` to `""`
before the operation so React can detect completion when the value transitions to non-empty
(per UI-SPEC memoryOpResult reset strategy).

NOTE: `m_MemorySystem` is already assigned in OnCreate via
`World.GetOrCreateSystemManaged<NarrativeMemorySystem>()` — no additional wiring needed.
